安装Astrometry.net

官方教程

系统准备

yum install出现Loaded plugins: fastestmirror, langpacks

1、修改插件配置文件

vi /etc/yum/pluginconf.d/fastestmirror.conf

将enable=1改为enable=0

2、修改yum配置文件

# vi /etc/yum.conf

将plugins=1改为plugins=0

然后yum clean all yum makecache

  • GNU build tools
  • Python 3
  • NetPBM
  • pyfits and cfitsio
  • 安装g++ gcc
